## ✨ Features

Digi Tools brings together the utilities developers and students reach for most — all in one place.

### 🔐 Encoders & Decoders

| Tool                     | Description                               |
| ------------------------ | ----------------------------------------- |
| Base64 Encoder / Decoder | Encode or decode Base64 strings instantly |
| URL Encoder / Decoder    | Safely encode and decode URL components   |

### 🔄 Converters

| Tool                | Description                                            |
| ------------------- | ------------------------------------------------------ |
| Image to Base64     | Convert images to embeddable Base64 strings            |
| Markdown to HTML    | Preview and export Markdown as HTML                    |
| Text Case Converter | Switch between UPPERCASE, lowercase, Title Case & more |

### ⚙️ Generators

| Tool                    | Description                                   |
| ----------------------- | --------------------------------------------- |
| Random Number Generator | Generate random numbers within a custom range |
| MD5 Hash Generator      | Generate MD5 hashes from any text input       |
| SHA Hash Generator      | Generate SHA-256 / SHA-512 hashes             |

### 🧪 Developer Utilities

| Tool               | Description                               |
| ------------------ | ----------------------------------------- |
| User Agent Info    | Inspect your browser's user agent string  |
| Timezone Detection | View your current timezone and UTC offset |
| IP Information     | Retrieve your public IP address details   |

## 🚀 Getting Started

### Prerequisites

- Node.js `v18+`
- npm or yarn

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/digi-tools.git
cd digi-tools

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```

The app will be available at `http://localhost:5173`.

## 🌍 Deployment

This project is deployed on **Netlify** with automatic CI/CD from GitHub.

To deploy your own instance:

1. Fork this repository
2. Go to [netlify.com](https://netlify.com) and create a new site
3. Connect your forked GitHub repository
4. Set build command: `npm run build` and publish directory: `dist`
5. Click **Deploy** — that's it!
