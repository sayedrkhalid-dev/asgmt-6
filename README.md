🚀 Digi Tools – All-in-One Online Utility Platform

🔗 Live Site: Digi Tools Web App

Digi Tools is a modern web application that provides a collection of useful online utilities for developers, students, and everyday users. It simplifies common tasks like encoding, decoding, generating hashes, and text transformations—all in one place.

📌 Features
🔐 Encoders & Decoders
Base64 Encoder / Decoder
URL Encoder / Decoder
🔄 Converters
Image to Base64
Markdown to HTML
Text Case Converters (Uppercase / Lowercase)
⚙️ Generators
Random Number Generator
MD5 & SHA Hash Generators
🧪 Developer Utilities
User Agent Info
Timezone Detection
IP Information
🎯 Clean UI
Simple and responsive design
Easy navigation
Fast performance
🛠️ Tech Stack
Frontend: React.js
Styling: Tailwind CSS / DaisyUI
Deployment: Netlify
📂 Project Structure
src/

│── assets/ # Images & icons

│── App.jsx # Main app component
│── main.jsx # Entry point
⚡ Getting Started

1. Clone the repository
   git clone https://github.com/your-username/digi-tools.git
   cd digi-tools
2. Install dependencies
   npm install
3. Run the project
   npm run dev
   🌍 Deployment

This project is deployed using Netlify, a platform that allows developers to build and deploy modern web applications with CI/CD and serverless features.

To deploy:

Push your code to GitHub
Connect repository to Netlify
Deploy with one click
